# Expense Tracker

## Project Overview
The Expense Tracker is a simple web application that helps users manage their daily expenses efficiently. 
Users can add, view, and delete expenses while tracking the total amount spent.

## How It Works
The user enters the expense name, amount, category, and date. 
When the "Add Expense" button is clicked:

- The expense is added to the table dynamically.
- The total amount is updated automatically.
- The form resets for the next entry.

Users can also delete any expense, and the total amount will update accordingly.

## Features
- Add new expenses
- Categorize expenses (Food, Transport, Entertainment, Other)
- View all expenses in a table format
- Automatic total calculation
- Delete expenses
- Real-time updates without page refresh

## Technologies Used
- HTML (Structure)
- CSS (Styling)
- JavaScript (Functionality & DOM manipulation)

## User Interaction
- The user fills the form and clicks "Add Expense".
- JavaScript handles the event and updates the UI dynamically.
- The application responds instantly without reloading the page.

## Future Improvements
- Add local storage to save expenses permanently.
- Add edit option for expenses.
- Add monthly or yearly reports.
- Add graphical representation (charts).

## Conclusion
The Expense Tracker project demonstrates the use of form handling, DOM manipulation, event listeners, and dynamic content updating using JavaScript. 
It is a beginner-friendly project that showcases practical web development skills.
